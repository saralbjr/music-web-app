# SoundWave - Music Player Web App

## Features

### Music Playback

- ✅ Play, pause, next, previous controls
- ✅ Seek bar with current time and duration display
- ✅ Volume slider with mute functionality
- ✅ Repeat modes (off, all, one)
- ✅ Shuffle playback
- ✅ Audio player with song information display

### Song Management

- ✅ Browse and search songs
- ✅ View individual song pages
- ✅ Add, edit, delete songs (admin)
- ✅ Upload MP3 files with cover images (admin)
- ✅ Song sorting and filtering
- ✅ Like/unlike songs
- ✅ Recently played songs

### Playlists

- ✅ Create, edit, and delete playlists
- ✅ Add and remove songs from playlists
- ✅ Custom playlist cover images
- ✅ Playlist management and organization

### User Features

- ✅ User registration and login system (email + password)
- ✅ User profile page
- ✅ Liked songs collection
- ✅ Personal library (playlists and liked songs)
- ✅ Search functionality across songs and categories

### Recommendations

- ✅ Content-based recommendation algorithm
- ✅ Mood-based music recommendations (Happy, Sad, Relaxed, Focused)
- ✅ Automatic mood detection from song categories and audio features
- ✅ Personalized song recommendations based on listening history

### Admin Features

- ✅ Admin dashboard with analytics
- ✅ User management (create, edit, delete users)
- ✅ Song management interface
- ✅ Playlist management
- ✅ System statistics and analytics

### Algorithms

- ✅ Merge Sort for efficient song sorting
- ✅ KMP (Knuth-Morris-Pratt) String Search for fast text searching
- ✅ Fisher-Yates Shuffle algorithm for random playback
- ✅ Content-based recommendation algorithm
- ✅ Mood detection algorithm (rule-based classification)
